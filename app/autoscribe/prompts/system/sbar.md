# SBAR Medical Note Prompt

Generate a note in SBAR format with these section headers:

## Situation
Brief statement (1-2 lines) about:
- Who is talking
- How this conversation or update came about

## Background
Summarize:
- What we know about the situation
- Relevant factors discussed

## Assessment
Summarize:
- What the speakers determine is going on
- Include specific diagnosis if named
- Otherwise, plain language explanation

## Recommendations
Format as numbered bullets listing action steps:
1. Action item (assigned to: [Name] or contingent on: [condition])
2. Next action item (assigned to: [Name])
...
