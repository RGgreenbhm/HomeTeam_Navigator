# Video Note Prompt

Please reference the attached document Transcript Text for context, which should contain the transcript of a primary care visit. Summarize the transcript as a medical office note. Begin the history section by referring to the patient by first name, which is included in the title of the reference document.

## Formatting Instructions
- Maintain text wrap and blank line spacing.
- Add an extra line at the beginning of the note with ":", followed by a line break.
- Insert a new line break in front of "-- " and "** " wherever they appear.

## Note Structure

REASON FOR VISIT: [on same line as section title, state why the patient is returning for visit; if the reason is not mentioned, then use "management of chronic problems" as default.]

HISTORY:
[Group information about interim health updates, details of new symptoms or medical concerns the patient may have, and status of chronic medical problem management with comments about related medications and tests into separate paragraphs.
- Do not number the paragraphs.
- Insert a line break before each new paragraph.
- Prioritize interim updates from ER visits and hospitalizations first, then updates from visits with specialists like cardiologist, pulmonologist, GI, and neurologist, then other specialists and new problems.
- For interim updates, include information about timing of specialist visits, any tests that were performed and any medication changes or other new recommendations from the specialist.
- For new problems, include details of when symptoms began, severity, factors that have made symptoms better or made symptoms worse.
- For medication updates, include clarification comments about how the patient has been taking medications prior to the visit, including the dose and time of day, if mentioned. Also, if mentioned, include comments about issues that prevented him or her from taking a medication as intended.]

HEALTH MAINTENANCE: [Note health maintenance plans. These include updates and recommendations related to cancer screenings, immunizations, fall risk (including use of assistive devices like cane, walker, or wheelchair), and caregivers who help with activities of daily living; if none mentioned, delete this section, including "HEALTH MAINTENANCE:" header.]

FAMILY HISTORY: [Note details about history and changes in health status of family members; if none mentioned, delete this section, including "FAMILY HISTORY:" header.]

SOCIAL HISTORY: [Note any status changes mentioned regarding marital status, employment status, alcohol use, cigarette use, and other social updates; if none mentioned, delete this section, including "SOCIAL HISTORY:" header.]

REVIEW OF SYSTEMS: [List symptoms that are mentioned as present on one line and symptoms that are mentioned as absent on another line. Include symptoms such as fever, chills, runny nose, sore throat, cough, shortness of breath, wheezing, chest pain, abdominal pain, nausea, vomiting, diarrhea, bloody stools, urinary frequency, pain with urination, urine odor, muscle aches, swollen joints, rashes, and enlarged lymph nodes.
- Add "-- " in front of "Present" and "Absent".
- Insert Line Break in front of each "-- "
- If none mentioned, then remove this line and delete "REVIEW OF SYSTEMS:" from the preceding paragraph.]

PHYSICAL EXAM:
-- VITAL SIGNS: [use "(See vital signs in chart section below)" as default; if measurements for temperature, blood pressure, pulse, oxygen saturation or weight are mentioned, include them here as well. Insert Line Break.]
-- GENERAL: [use "pleasant, well developed adult" as default, if gender is mentioned at beginning of transcript, include "male" or "female" next; then describe as "appearing in usual state of health" vs. "appearing unwell;" note if the patient is using a cane, walker, or wheelchair to get to the visit (omit if not mentioned); note whether the patient presented alone vs. with a family member or friend. Insert Line Break.]
-- HEAD: [list findings related to the head and scalp here; if none mentioned, add "normocephalic, atraumatic". Insert Line Break.]
-- EYES: [list findings related to the eyes here; if none mentioned, add "normal, sclera non-icteric". Insert Line Break.]
-- EARS: [If mentioned, list findings related to the ear, external ear canals, and tympanic membrane (ear drums) here; If none of these are mentioned add "normal, without significant cerumen or abnormalities of tympanic membranes". Insert Line Break.]
-- NOSE: [If mentioned, list findings related to the nose, nostrils, and sinuses here; if none of these are mentioned add "normal". Insert Line Break.]
-- MOUTH: [If mentioned, list findings related to the mouth, lips, tongue, teeth, and buccal mucosa here; if none of these are mentioned add "normal, no lesions". Insert Line Break.]
-- THROAT: [If mentioned, list findings related to the posterior pharynx, tonsils, and throat here; if none of these are mentioned add "clear". Insert Line Break.]
-- CARDIOVASCULAR: [list findings related to "heart", "heart rate", "heart sounds", "heart rhythm", "leg swelling", "edema", "carotid arteries", and location of defibrillator or heart monitors on body here; if none mentioned, add "RRR". Insert Line Break.]
-- PULMONARY: [list findings related to respirations, breathing, wheezing, and focal changes in breath sounds on lung exam here; if none mentioned add "respirations are non-labored". Insert Line Break.]
-- GASTROINTESTINAL: [list findings related to the "abdomen," "belly," "stomach," and "abdominal surgical scars" here; if none of these are mentioned add "soft, nontender". Insert Line Break.]
-- DERMATOLOGIC: [list findings related to skin, bruising, hair, nails, and cellulitis here; if none are mentioned, add "skin warm, dry, and without rashes". Insert Line Break.]
-- MUSCULOSKELETAL: [list findings related to joints, muscles, and range of motion here; if none mentioned, add "normal muscle bulk and tone, without hot or swollen joints". Insert Line Break.]
-- NEUROLOGIC: [list findings related to cognitive status, numbness, weakness, gait, and balance here; if none mentioned, add "alert with normal interaction; grossly intact". Insert Line Break.]
-- PSYCHIATRIC: [list findings related to mood and affect here; if none mentioned, add "normal mood and affect, thought processes are logical and goal directed". Insert Line Break.]

MEDICAL DECISION MAKING:
[Group information about management decisions together with the related diagnoses (problems) in separate paragraphs. Do not number the paragraphs. Insert a line break before each new paragraph.
For each diagnosis, first include an assessment of what seems to be causing or contributing to this diagnosis and what needs to happen to make it better or to keep it controlled, then summarize recommendations including diagnostic plans such as lab tests, radiology (imaging orders), specialist referrals, and treatment options including patient education, pharmacotherapy if any, in office treatments like medication injections, nebulizer treatments or referrals for other planned therapeutic procedures. Include comments about risks, benefits and alternatives if mentioned for the current diagnosis.
Repeat for all diagnoses or problems mentioned in the transcript.]

FOLLOW-UP:
-- [List plans for when the patient is to return for next visit or re-evaluation with us. If plans for when the patient is scheduled to see any specialists are also mentioned, include those appointment details in chronological order on new lines below, with "-- " in front of each new line.]

VISIT DETAILS:
