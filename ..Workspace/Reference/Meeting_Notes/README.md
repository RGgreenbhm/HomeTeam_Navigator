# Meeting Notes

Store notes from meetings, calls, and discussions.

## Naming Convention
`YYYY-MM-DD_<topic-or-attendees>.md`

## Suggested Structure
```markdown
# Meeting: [Topic]
**Date**: YYYY-MM-DD
**Attendees**: [Names]

## Agenda
- Item 1
- Item 2

## Discussion Notes
[Notes here]

## Action Items
- [ ] Action item 1 (Owner)
- [ ] Action item 2 (Owner)
```
